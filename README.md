En este proyecto lo que buscamos trabajar es la parte tecnologica de la Biblioteca del colegio Tecnico N1 de Brandsen, este equipo está conformado por estudiantes del grupo 1 de 7mo 5ta. De la parte de código se está encargando Benicio Caraccio, Carolina Cardoso por su parte está trabajando con los diagramas y por ultimo, Ariel Erretegui se hace cargo de la curva S.

Nuestro objetivo es desarrollar una App Web interactiva que administrar las PC´s mediante una interfaz simple y facil de utilizar

Tenemos pensado llevar a cabo este proyecto con nuestras ideas, y las que vayamos aprendiendo mediante hagamos este trabajo. Por el momento estamos usando la tecnología React + JavaScript, sin embargo en un futuro tenemos pensado incluir Node.js + Express para el Backend y MySQL para la base de datos.

Hasta el momento hemos avanzado bastante en la parte de App.jsx, donde podemos encontrar todo nuestro codigo dedicado a la reserva de las PC´s y cancelación de la misma.

Las funcionalidades que nos brinda esta app son, visualización del estado de las computadoras, reserva de computadoras disponibles,cancelación de reservas y gestión de disponibilidad en tiempo real.
